How to play

The goal of this currently unnamed game is to create the funniest meme you can. You will be shown a series of templates which you have to come up with a caption for. Once everyone has submitted their caption or the timer runs out all of the captions will be displayed and you have to vote on which you think is the best. Once everyone has voted the winning meme will be revealed and scores will be adjusted accordingly(10 points per vote). Once the alloted number of rounds have passed the game is over and the person with the most votes wins!

To start the game have one of the players in your group create a room by clicking the button create room. Once the room has been created players can join by entering the room ID and clicking Join room. Once all the players are in the room one of the players can select the number of rounds(5, 10 or 20) then press the start button. Players cannot join after the game has started. Also please don't use the same username as someone else in the room, it will get a little confused.

When the templates are displayed they will show a set of numbers corresponding to inputs. To change between inputs separate your caption by commas. For example if the template has 3 text fields you could write Thank, you, for playing! "Thank" would be displayed in the first field, "you" in the second and "for playing!" in the third.
